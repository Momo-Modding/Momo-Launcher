# Momo Launcher

A lightweight, game launcher for Momo.

## Features
* **Instances:** Create different game configurations with different mods.
* **Mod Manager:** Install, update and remove mods easily with the mod manager.
* **Drag-and-Drop Local Installs:** Drop any `.zip` mod file directly into the app to install it.
* **Playtime Tracking:** Tracks your hours per instance.

## Installation (For Players)
1. Go to the [Releases page](https://github.com/Momo-Modding/Momo-Launcher/releases).
2. Download the latest release.
3. Run the installer and launch the app!
